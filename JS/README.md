# JS — Concise Data Structures & Algorithms Examples

Short, runnable JavaScript examples for each topic from the main app.

| File                      | Topic                                    |
| ------------------------- | ---------------------------------------- |
| `01-sameFrequency.js`     | Same digit frequency between two numbers |
| `02-recursion.js`         | Countdown, factorial, collect odds       |
| `03-classes.js`           | Student, Point (instance + static)       |
| `04-singleLinkedList.js`  | SLL: push, pop, get, reverse             |
| `05-doubleLinkedList.js`  | DLL: push, pop, shift, unshift, get      |
| `06-stack.js`             | Stack (LIFO)                             |
| `07-queue.js`             | Queue (FIFO)                             |
| `08-binarySearchTree.js`  | BST: insert, find, BFS, DFS              |
| `09-maxBinaryHeap.js`     | Max heap: insert, extractMax             |
| `10-priorityQueue.js`     | Min-heap priority queue                  |
| `11-hashTable.js`         | Hash table: set, get, keys, values       |
| `12-graph.js`             | Graph: addVertex, addEdge, DFT, BFT      |
| `13-bigO.js`              | Count evens, O(1)/O(n)/O(n²)             |
| `14-sequenceADT.js`       | Remove duplicate, isPermutation          |
| `15-recursionAdvanced.js` | Fibonacci, isEven, power                 |
| `16-examHelpers.js`       | isEvenMutual, sum, sumFirst              |

**Run one file:**

```bash
node JS/01-sameFrequency.js
```

**Run all:**

```bash
node JS/run-all.js
```
